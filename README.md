These are the exposure files for the 2020 Java exposure model developed by Weber et al. 2026. The paper contains all of the references for the input datasets that were used in the development of this exposure model. Any use of the data must reference the paper:
A building exposure model for Java, Indonesia for use in Seismic Risk Assessment
Weber, R.; Cummins, P.; Silva, V. and Edwards, M. 2026
Bulletin of Earthquake Engineering (Accepted for publication)
The titles of the files are self explanatory for the building uses contained within. 
M1 is masonry houses using mappin scheme 1 and M2 is masonry houses using mapping scheme 2 
Each file contains a unique id, lon, lat, taxonomy, number of buildings, strucutral value, AMD1, ADM2, ADM3,ADM4, a unique desaID, use, and an occupancy of night and/or day
